Settings

Files in the settings folder exist for the sole purpose of setting variables that are used in the rest of the codebase. There shouldn't be any code that is not a variable declaration.

_s-borders.css

:root {
    --borderWidth: 1px;
    --borderColor: var(--color-text);             /* #141515 */
    --borderColor-light: var(--color-text-light); /* #81858B */
    --borderRadius: 4px;
    --borderRadius-button: 4px;
}

_s-breakpoints.css

There are 7 breakpoints built in and 13 custom media queries (6 min and 6 max and one for the site-container width) defined around them. They are defined in em units which allows them to adapt to users who set a different base font size in their browser or zoom in/out, as long as there is no explicit (px) font size set for the body/html elements. 

@custom-media --screen-xs-min (min-width: 25em);             /* 400px  */
@custom-media --screen-sm-min (min-width: 32em);             /* 512px  */
@custom-media --screen-md-min (min-width: 48em);             /* 768px  */
@custom-media --screen-lg-min (min-width: 64em);             /* 1024px */
@custom-media --screen-xl-min (min-width: 90em);             /* 1440px */
@custom-media --screen-xxl-min (min-width: 120em);           /* 1920px */

@custom-media --screen-xxs-max (max-width: 24.9375em);       /* 399px  */
@custom-media --screen-xs-max (max-width: 31.9375em);        /* 511px  */
@custom-media --screen-sm-max (max-width: 47.9375em);        /* 767px  */
@custom-media --screen-md-max (max-width: 63.9375em);        /* 1023px */
@custom-media --screen-lg-max (max-width: 89.9375em);        /* 1439px */
@custom-media --screen-xl-max (max-width: 119.9375em);       /* 1919px */

@custom-media --screen-site-container (max-width: 104.4rem); /* 1024px */

_type-scale.css

The variables --ratio (default: 1.25) and --mobile-ratio (default: 1.2) are used to define the typographic scale used in headings and font-size classes. These classes will adjust their type scale for mobile sizes to optimize for all screen sizes.

Check out Type Scale to learn more about modular scales and see them in action.

_s-typography.css

All font-size variables are based off of --base-font-rem --mobile-base-font-rem and the variables from _s-type-scale.css and assume that the font-size is set to 62.5% on the html tag. 

:root {
    --base-font-rem: calc(1.6 * 1rem);
    --mobile-base-font-rem: calc(1.4 * 1rem);

    --lineHeight: 1.6;

    --fontSize-n1: calc(var(--base-font-rem) * var(--scale-n1));
    --fontSize-1:  calc(var(--base-font-rem) * var(--scale-0));
    --fontSize-2:  calc(var(--base-font-rem) * var(--scale-1));
    --fontSize-3:  calc(var(--base-font-rem) * var(--scale-2));
    --fontSize-4:  calc(var(--base-font-rem) * var(--scale-3));
    --fontSize-5:  calc(var(--base-font-rem) * var(--scale-4));
    --fontSize-6:  calc(var(--base-font-rem) * var(--scale-5));
    --fontSize-7:  calc(var(--base-font-rem) * var(--scale-6));
    --fontSize-8:  calc(var(--base-font-rem) * var(--scale-7));

    --mobile-fontSize-n1: calc(var(--mobile-base-font-rem) * var(--mobile-scale-n1));
    --mobile-fontSize-1:  calc(var(--mobile-base-font-rem) * var(--mobile-scale-0));
    --mobile-fontSize-2:  calc(var(--mobile-base-font-rem) * var(--mobile-scale-1));
    --mobile-fontSize-3:  calc(var(--mobile-base-font-rem) * var(--mobile-scale-2));
    --mobile-fontSize-4:  calc(var(--mobile-base-font-rem) * var(--mobile-scale-3));
    --mobile-fontSize-5:  calc(var(--mobile-base-font-rem) * var(--mobile-scale-4));
    --mobile-fontSize-6:  calc(var(--mobile-base-font-rem) * var(--mobile-scale-5));
    --mobile-fontSize-7:  calc(var(--mobile-base-font-rem) * var(--mobile-scale-6));
    --mobile-fontSize-8:  calc(var(--mobile-base-font-rem) * var(--mobile-scale-7));

    --fontWeight-thin: 300;
    --fontWeight-normal: 400;
