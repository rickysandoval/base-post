What is BasePost?

BasePost is intended to be used as a starting point for CSS developers building medium to large projects using PostCSS. It's not a UI framework, just a tool to help speed up the development process while still encouraging high quality, maintainable codebases.

Flexible

BasePost is made to mold to your wishes. Quickly tweak the feel of an entire project by changing a few variables.

Responsive

Breakpoints respond to not only screen width, but also zooming and user-set font size. Type scales automatically adjust for smaller and larger screens.

Lightweight

Structure

BasePost incorporates ideas from ITCSS, OOCSS, and BEM in it's organization. It's built with the CSS cascade, source order and specificity in mind. Although it comes pre-structured, it is still easy to chose what direction to take your project in, whether you feel more comfortable with OOCSS, Atomic CSS, SMACSS, or whatever else.
