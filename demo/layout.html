Grid

I chose not to implement a custom grid, because well, there's so many different grid systems out there and everyone has their favorites. I'm including the flexbox based Gridlex because the syntax is simple and powerful and isn't like learning a mini framework.

I manually imported the source and edited it to work with PreCSS and BasePost's settings, but you can swap it out with whatever you want.

Check out the documentation for Gridlex if you don't want to use your own.
