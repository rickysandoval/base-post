Settings

Files in the settings folder exist for the sole purpose of setting variables that are used in the rest of the codebase. There shouldn't be any code that is not a variable declaration.

Breakpoints

There are 7 breakpoint "zones" built in: xxs, xs, sm, md, lg, and xxl. There are 12 custom media queries (6 min and 6 max) defined around them. They are defined in em units which allows them to adapt to users who set a different base font size in their browser, as long as there is no explicit (px) font size set for the body/html elements.
